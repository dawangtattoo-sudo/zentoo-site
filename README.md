# ZENTOO / 针武堂刺青 本地工作流

这是 ZENTOO / 针武堂刺青的本地品牌内容工作流项目，用于沉淀品牌原则、小红书内容模板、海报规则、GEO 关键词、NAS 管理方案与官网首页规划。

## 品牌核心

- 品牌方向：高级、克制、现代、品牌感、多人纹身店整体气质
- 品牌态度语：不要徘徊在别人的看法里
- 英文表达：TATTOO YOUR FREEDOM
- 内容原则：不写客户隐私、不写电话、不写真实报价

## 当前目录

- `brand/brand-principles.md`：品牌原则与表达边界
- `xiaohongshu/content-template.md`：小红书内容模板
- `posters/poster-rules.md`：海报视觉与文案规则
- `geo/geo-keywords.md`：本地搜索与 GEO 关键词方向
- `nas-workflow/nas-plan.md`：NAS 文件归档与协作方案
- `website/homepage-plan.md`：官网首页结构规划

## 使用方式

每次新增内容前，先确认是否符合品牌核心。所有对外发布内容都应保持克制、清晰、有质感，避免夸张营销、低价刺激、过度承诺或暴露客户隐私。
