# ZENTOO / 针武堂刺青发布前检查清单

本文件用于每次发布小红书、海报、官网内容前检查。发布前建议逐条确认，确保内容符合 ZENTOO / 针武堂刺青的品牌标准。

## 固定信息

- 品牌：针武堂刺青 / ZENTOO
- 成立时间：2009 年
- 团队：12 位纹身师共同构成的专业团队
- 方向：高级、克制、现代、多人纹身店整体气质
- 常规纹身业务：覆盖郑州、河南及周边
- 疤痕遮盖业务：可面向全国
- 品牌态度语：不要徘徊在别人的看法里
- 对应英文：TATTOO YOUR FREEDOM

## 一、品牌气质检查

- [ ] 内容是否符合高级、克制、现代的方向？
- [ ] 内容是否体现多人纹身店的整体气质？
- [ ] 内容是否避免过度热闹、低价、江湖或网红感？
- [ ] 内容是否有清楚的审美判断，而不是单纯推销？
- [ ] 内容是否让人感到专业、稳定、可信？

## 二、品牌识别检查

- [ ] 内容中是否出现 ZENTOO 或针武堂刺青？
- [ ] 如果是品牌主内容，是否出现「不要徘徊在别人的看法里」？
- [ ] 如果需要英文表达，是否严格使用「TATTOO YOUR FREEDOM」？
- [ ] 是否没有随意改写品牌态度语？
- [ ] 是否没有把品牌表达写得太散、太像普通纹身账号？

## 三、隐私与敏感信息检查

- [ ] 是否没有客户真实姓名？
- [ ] 是否没有客户联系方式？
- [ ] 是否没有客户聊天记录？
- [ ] 是否没有电话？
- [ ] 是否没有真实报价？
- [ ] 是否没有未经授权的人像或可识别身份信息？
- [ ] 疤痕遮盖内容是否没有过度描述客户私人经历？

## 四、业务信息检查

- [ ] 常规纹身内容是否可以自然覆盖郑州、河南及周边客户？
- [ ] 疤痕遮盖内容是否可以自然说明可面向全国沟通？
- [ ] 是否没有夸大服务范围？
- [ ] 是否没有承诺“百分百遮住”“绝对满意”“无痛”等绝对结果？
- [ ] 是否没有突出单个纹身师个人 IP，而是保持团队品牌感？

## 五、小红书平台阅读检查

- [ ] 标题是否简洁、有态度，但不夸张？
- [ ] 开头是否能让人继续读下去？
- [ ] 正文是否分段清楚，手机上容易阅读？
- [ ] 是否适合收藏、转发或作为纹身前参考？
- [ ] 标签是否和内容相关，没有乱堆关键词？
- [ ] 是否避免“低价引流”“标题党”“过度情绪化”？

## 六、海报发布检查

- [ ] 主标题是否足够简洁？
- [ ] 画面是否有留白？
- [ ] 是否没有塞入太多文字？
- [ ] 是否突出作品、空间或品牌气质？
- [ ] 是否没有电话、真实报价和客户隐私？
- [ ] 是否适合小红书、朋友圈或门店屏幕使用？

## 七、官网发布检查

- [ ] 首页或页面内容是否能清楚说明 ZENTOO / 针武堂刺青是谁？
- [ ] 是否体现 2009 年成立和 12 位纹身师团队？
- [ ] 是否说明常规纹身服务郑州、河南及周边？
- [ ] 是否说明疤痕遮盖业务可面向全国？
- [ ] 是否没有写真实报价和电话？
- [ ] 是否符合长期品牌门面的质感？
