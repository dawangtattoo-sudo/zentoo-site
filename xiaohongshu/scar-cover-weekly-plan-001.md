# ZENTOO疤痕遮盖 7天内容计划 001

本计划用于 ZENTOO 疤痕遮盖 / 疤痕纹身美化专项号。它不服务蓝 V 店铺品牌号，也不服务作品视频号。

本周内容必须遵守 `xiaohongshu/scar-cover-account-system.md`：主品牌识别使用 ZENTOO，不使用“针武堂刺青”中文字；主打疤痕遮盖、疤痕纹身美化、疤痕视觉重整；不安排胎记主题；不做旧纹身遮盖，不做失败纹身遮盖；不医疗化，不绝对承诺，不低价促销。

## 1. 本周目标

本周目标不是追求爆量，而是建立账号稳定内容结构。

目标包括：

- 让主页更像一个专业疤痕遮盖专项账号
- 提高封面点击率
- 提高收藏率
- 增加用户对“方案效果”和“真实案例”的理解
- 建立 ZENTOO疤痕遮盖 的账号记忆点
- 让用户知道：疤痕遮盖不是简单盖住，而是重新整理身体痕迹和图案之间的视觉关系

## 2. 本周内容比例

本周 7 篇内容按以下结构规划：

1. 真实案例 2 篇
2. 方案效果 / 示范案例 2 篇
3. 判断科普 1 篇
4. 情绪价值 1 篇
5. 咨询准备 / 转化内容 1 篇

内容作用：

- 真实案例用于建立信任
- 方案效果用于补充真实案例不足，展示不同位置的设计可能
- 判断科普用于建立专业感和收藏价值
- 情绪价值用于建立共鸣
- 咨询准备用于促进用户私信咨询

### 推荐标题

手腕怎么做

### 正文文案

手腕线状疤痕，是很常见的咨询位置。

但它不一定适合直接“盖满”。

更重要的是，顺着疤痕的走向，重新安排图案和皮肤之间的视觉关系。

如果疤痕呈线状，可以考虑让图案沿着它的方向延展，比如细线条、植物枝叶、轻量图形、局部黑灰层次。

重点不是让图案变得很重，而是让原本突兀的位置，变成画面里自然的一部分。

一般建议疤痕状态稳定一年以上，再考虑纹身美化。但具体是否适合，需要结合实际皮肤状态判断。

这篇是方案效果 / 设计参考，不是真实客户案例。

### 内容类型

### 推荐标题

### 正文文案

不是所有疤痕都适合马上做纹身美化。

也不是所有疤痕都适合同一种设计方式。

判断前，至少要看这几个方向。

1. 疤痕状态是否稳定  
一般建议稳定一年以上，再考虑纹身美化。

2. 疤痕位置是否适合图案延展  
有些位置适合顺着身体结构设计，有些位置需要更克制。

3. 疤痕面积和走向  
面积、长短、方向都会影响图案选择。

4. 皮肤状态  
是否适合操作，需要结合实际皮肤状态判断。

疤痕遮盖不是简单盖住，而是重新整理皮肤上的视觉关系。

4. 第 4 页：还要看走向

想做疤痕遮盖，咨询前可以先准备这些信息。

信息越清楚，越方便初步判断设计方向。

一般建议状态稳定一年以上，再考虑纹身美化。

4. 想要的风格  
可以准备喜欢的图案、花枝、线条、黑灰或其他参考方向。

5. 预算范围和所在城市  
方便沟通设计复杂度和后续安排。

疤痕遮盖不是简单盖住，而是重新整理身体痕迹和图案之间的视觉关系。具体是否适合，需要结合实际皮肤状态判断。

## 5. 主页统一提醒

本周主页视觉提醒：

- 封面尽量使用统一色系
- 主色建议：米白、浅灰、低饱和绿、浅咖、柔和粉
- 避免强烈蓝黑大字报风格
- 封面文字不宜过多
- 每张图默认加入 ZENTOO
- 真实案例和方案效果要用不同小角标区分
- 真实案例角标：真实案例 / 覆盖记录
- 方案效果角标：方案效果 / 设计参考
- 不要把方案效果写成真实客户案例
