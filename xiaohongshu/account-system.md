# ZENTOO / 针武堂小红书多账号运营系统

本文件用于规划 ZENTOO / 针武堂当前正在运营的三个小红书账号体系。它不推翻当前 GitHub / Codex 工作流，也不重建项目。当前 `zentoo-site` 是 ZENTOO / 针武堂刺青的 AI 工作流总仓库；本文件是在现有 `AGENTS.md`、`brand/`、`xiaohongshu/`、`posters/` 规则基础上，补充小红书多账号分工规则。

## 1. 总原则

当前小红书运营包含三个账号：

1. 蓝 V 店铺品牌号
2. 作品视频号
3. ZENTOO 疤痕遮盖专项号

三个账号都属于 ZENTOO / 针武堂纹身运营体系，但内容定位不同，不能混在一起使用同一套内容。

共同原则：

- 品牌底层审美保持高级、克制、现代、专业
- 内容不能低价促销感
- 不能江湖感
- 不能夸张网红感
- 不加入客户隐私、电话、真实报价
- 视觉要有设计感和统一排版意识
- 作品是核心，设计排版是差异化
- 不为了发内容而牺牲质量

## 2. 账号一：蓝 V 店铺品牌号

### 账号定位

蓝 V 账号是 ZENTOO / 针武堂刺青 的店铺品牌号。

它不是纯作品号，也不是纯品牌理念号，而是用于展示：

- 店内日常
- 多人纹身店整体气质
- 纹身师工作和生活的一面
- 店铺空间
- 团队状态
- 部分作品更新
- 本周工作内容
- 真实营业状态

核心目标：

让用户除了看到作品之外，也能理解这是一家由多位纹身师共同组成的专业纹身店，有空间、有团队、有工作状态、有真实日常。

### 内容重点

蓝 V 账号适合发布：

- 店内日常视频
- 多人纹身师工作状态
- 店内空间氛围
- 本周作品图文总结
- 本周作品视频总结
- 纹身手稿 / 设计过程
- 单个作品审美解析
- 新手纹身科普
- 店铺品牌态度
- 团队状态和工作生活侧面

### 蓝 V 内容比例建议

一周内容储备池建议：

- 店内日常 / 工作状态视频：2 篇
- 作品图文 / 作品总结：2 篇
- 店内空间 / 氛围视频：1 篇
- 手稿 / 设计过程：1 篇
- 新手科普 / 品牌态度：1 篇

实际发布节奏：

- 基础：每周 3-5 篇
- 素材充足时可短期日更
- 不为日更牺牲封面质量和内容差异

### 蓝 V 视觉规则

- 封面要有主页整体规划
- 作品仍然重要，但要通过排版和文字设计呈现差异化
- 画面要像“作品编辑化呈现”，不是普通纹身店作品流水账
- 可使用 “ZENTOO / 针武堂刺青”
- 封面和图文中应出现 ZENTOO，并在合适位置出现“针武堂刺青”
- 标题短、准、有判断感，默认 4-9 字

## 3. 账号二：作品视频号

### 账号定位

作品视频号只发纹身作品，并且全部以视频形式展示作品。

这个账号不承担店铺日常、品牌理念、团队生活内容。它的目标是用视频展示作品质感、完成效果、身体位置、局部细节和风格完成度。

### 内容重点

作品视频号适合发布：

- 单个作品完成视频
- 作品局部扫拍
- 作品质感视频
- 身体位置关系展示
- 本周作品视频合集
- 不同风格作品视频
- 黑灰 / 线条 / 东方 / 写实等作品视频分类
- 手部、背部、腿部、肩颈等不同位置完成效果

### 作品视频号内容规则

- 只发作品相关视频
- 不发店内日常
- 不发过多品牌理念长文
- 不做纹身师个人生活内容
- 不做低价促销
- 不做夸张口播
- 视频重点是作品本身
- 封面仍然要有设计感和统一排版
- 每条视频要突出一个作品重点，例如位置、线条、结构、黑灰层次、身体关系

### 作品视频号封面规则

- 视频封面不能随便截帧
- 选择作品最清晰、最有质感的一帧
- 标题短，尽量 4-9 字
- 可使用：
  - 本周作品
  - 作品记录
  - 位置很重要
  - 线条要克制
  - 黑灰层次
  - 作品要耐看
- 视觉重点是作品，不让文字抢作品
- 品牌识别可使用 ZENTOO，也可根据账号定位弱化中文品牌名

## 4. 账号三：ZENTOO 疤痕遮盖专项号

### 账号定位

这是 ZENTOO 名下的疤痕遮盖专项账号。

重要边界：

- 只做疤痕遮盖
- 不做旧纹身遮盖
- 不做失败纹身遮盖
- 不混淆为旧纹身 cover up 账号
- 账号视觉中不使用“针武堂刺青”中文字
- 以 ZENTOO 作为主要品牌识别

### 内容重点

疤痕遮盖专项号适合发布：

- 疤痕遮盖案例
- 疤痕遮盖设计逻辑
- 皮肤状态判断
- 疤痕位置、走向、颜色、面积对设计的影响
- 设计如何重新整理皮肤视觉关系
- 遮盖前沟通注意事项
- 疤痕遮盖科普
- 情绪修复和身体表达
- 全国范围疤痕遮盖咨询内容

### 疤痕遮盖专项号禁区

禁止：

- 发布旧纹身遮盖内容
- 发布失败纹身遮盖内容
- 使用“旧纹身修改”“失败纹身遮盖”“cover up 旧纹身”等表达
- 用猎奇方式展示疤痕
- 夸张承诺“完全看不出”
- 恐吓式营销
- 暴露客户隐私
- 展示未经授权的身体隐私部位
- 使用低价促销话术
- 把疤痕遮盖做成低级前后对比冲击

### 疤痕遮盖账号视觉规则

- 主品牌识别使用 ZENTOO
- 不出现“针武堂刺青”中文字
- 视觉更克制、更专业、更安静
- 重点表达设计判断，不做猎奇冲击
- 可以强调：
  - scar cover
  - scar tattoo
  - skin story
  - body freedom
  - TATTOO YOUR FREEDOM
- 中文表达要温和、专业，不刺激用户伤痛
- 不能用过度煽情和恐吓式标题

## 5. 三个账号的关系

三个账号不是互相替代，而是分工：

蓝 V 店铺品牌号：

负责建立用户对店铺整体的了解、信任和品牌感。

作品视频号：

负责高频展示作品实力，用作品视频吸引喜欢纹身风格的用户。

ZENTOO 疤痕遮盖专项号：

负责全国范围疤痕遮盖业务，建立专业、克制、可信的专项形象。

## 7. 使用规则

### 账号简称使用规则

以后用户在对话中可以直接使用以下账号简称：

#### 1. 疤痕遮盖账号

对应：ZENTOO 疤痕遮盖专项账号。

默认规则：

- 只服务疤痕遮盖、疤痕纹身美化、疤痕视觉重整
- 胎记不是主要宣传点
- 不做旧纹身遮盖
- 不做失败纹身遮盖
- 不出现“针武堂刺青”中文字
- 默认使用 ZENTOO
- 不医疗化
- 不承诺百分百覆盖
- 方案效果不能冒充真实案例

#### 2. 蓝V账号

对应：ZENTOO / 针武堂刺青官方蓝 V 店铺品牌号。

默认规则：

- 店内日常
- 多人纹身店工作状态
- 店铺空间
- 团队气质
- 部分作品展示
- 可以出现“ZENTOO / 针武堂刺青”
- 强调多人纹身店整体品牌感，不突出单个纹身师个人 IP

#### 3. 作品账号

对应：ZENTOO / 针武堂体系下的作品视频号。

默认规则：

- 只发纹身作品
- 全部以视频形式展示
- 不发店内日常
- 不做品牌理念长文
- 重点展示作品完成度、质感、位置、风格和身体关系

以后如果用户只说“疤痕遮盖账号 / 蓝V账号 / 作品账号”，就按照对应账号规则处理。

以后让 Codex 生成小红书内容时，必须先确认是哪个账号：

- 蓝 V 店铺品牌号
- 作品视频号
- ZENTOO 疤痕遮盖专项号

不能默认所有内容都按蓝 V 账号生成。

如果用户没有说明账号，Codex 应先根据主题判断：

- 店内日常、团队、空间、品牌感：优先蓝 V
- 纯作品视频：优先作品视频号
- 疤痕遮盖：优先 ZENTOO 疤痕遮盖专项号

如果主题跨账号，例如“疤痕遮盖案例视频”或“本周作品总结”，应先判断内容重点：

- 重点是疤痕遮盖专业判断：归入 ZENTOO 疤痕遮盖专项号
- 重点是纯作品视频质感：归入作品视频号
- 重点是店铺整体状态和品牌信任：归入蓝 V 店铺品牌号
