# ZENTOO疤痕遮盖 内容出图需求模板

本文件用于 ZENTOO 疤痕遮盖 / 疤痕纹身美化专项号后续快速提出小红书封面、图文和出图需求。使用时应遵守 `xiaohongshu/scar-cover-account-system.md`、`xiaohongshu/scar-cover-weekly-plan-001.md`、`xiaohongshu/scar-cover-homepage-layout.md` 的规则。

## 1. 模板用途

本模板用于 ZENTOO疤痕遮盖账号后续生成：

- 小红书封面
- 方案效果图
- 真实案例图文
- 判断科普图文
- 情绪价值图文
- 咨询准备图文
- 6页图文笔记
- 主页统一内容

使用本模板时，用户只需要填写疤痕位置、疤痕类型、想要风格、封面要求等信息，即可快速生成更准确的内容方案。

## 3. 使用提醒

- 如果选择“真实案例”，必须使用真实客户案例或真实完成作品，不要把方案效果写成真实案例。
- 如果选择“方案效果 / 设计参考”，必须明确标注“方案效果”“设计参考”或“方案参考”。
- 默认主品牌识别只使用简洁文字 “ZENTOO”，不使用“针武堂刺青”中文字。
- 不要在 ZENTOO 旁边加入红色印章、小圆章、篆刻章、复杂 logo 图形或装饰性图标。
- 不写客户隐私、客户电话、真实报价。
- 不做旧纹身遮盖，不做失败纹身遮盖。
- 不使用“百分百遮住”“完全看不出”“无痕修复”“治疗”“诊断”等表达。
- 语气保持温和、克制、专业，有情绪价值但不过度煽情。
