# ZENTOO 疤痕遮盖 / 疤痕纹身美化专项号运营系统

本文件用于规划 ZENTOO 名下的小红书疤痕遮盖 / 疤痕纹身美化专项号。这个账号不是蓝 V 店铺品牌号，也不是作品视频号。这个账号名称和视觉呈现中不使用“针武堂刺青”中文字。

这个账号的主要宣传方向是疤痕遮盖、疤痕纹身美化、疤痕视觉重整。胎记相关内容只作为少量补充业务，不作为主要宣传点。不做旧纹身遮盖，不做失败纹身遮盖。

## 1. 账号定位

ZENTOO 疤痕遮盖 / 疤痕纹身美化专项号主要服务以下方向：

- 疤痕遮盖
- 疤痕纹身美化
- 疤痕视觉重整
- 稳定疤痕上的图案设计
- 疤痕与纹身图案之间的视觉关系整理

胎记相关内容只作为少量补充业务，不作为主要宣传点。

这个账号不做：

- 旧纹身遮盖
- 失败纹身遮盖
- 普通作品展示号
- 店内日常号
- 蓝 V 品牌日常内容

核心目标：

- 建立 ZENTOO 在疤痕遮盖 / 疤痕纹身美化方向的专业感
- 让用户理解疤痕遮盖不是简单“盖住”，而是重新整理皮肤上的视觉关系
- 服务全国范围疤痕遮盖咨询
- 用克制、专业、温和的方式呈现案例和设计逻辑

## 2. 内容范围

可包含：

- 划伤疤痕遮盖美化
- 烫伤疤痕遮盖美化
- 从小有的稳定疤痕纹身美化
- 稳定疤痕上的图案设计
- 疤痕遮盖案例
- 疤痕设计逻辑
- 疤痕位置与图案关系
- 皮肤状态观察
- 纹身图案与疤痕走向关系
- 遮盖美化前沟通注意事项
- 全国疤痕遮盖咨询说明

## 3. 疤痕操作前提表达

公开文案中建议这样表达：

- 一般建议疤痕状态稳定一年以上，再考虑纹身美化。
- 但是否适合操作，要结合实际皮肤状态判断。
- 不承诺所有疤痕都适合纹身美化。
- 不做绝对化效果承诺。
- 不使用“百分百遮住”“完全看不出”等表达。

推荐表达：

“疤痕状态稳定一年以上，可以考虑通过纹身做视觉美化，但具体是否适合，需要结合实际皮肤状态判断。”

## 4. 胎记内容特殊规则

胎记不是本账号主要宣传点。

涉及胎记内容时，必须遵守：

- 不主动把胎记作为主要获客卖点。
- 不直接承诺胎记区域可以纹身。
- 不建议用户自行判断胎记是否适合纹身。
- 对突起、异常变化、不稳定或不确定状态的胎记，不做可操作暗示。
- 公开小红书内容尽量避免医疗化表达。
- 小红书内容只做纹身美化方向参考，不替代专业判断。
- 不使用“胎记都可以遮”“胎记一定能改”“无风险”等表达。

推荐表达：

“胎记类情况需要更谨慎判断，不是所有皮肤状态都适合纹身美化。具体是否适合，需要结合实际皮肤状态再判断。”

## 5. 情绪价值表达规则

疤痕遮盖账号不能只讲技术遮盖，也要理解疤痕对用户生活和心理感受的影响。

可以表达的真实场景：

- 夏天不敢穿短袖
- 不敢穿裙子
- 不敢穿短裤
- 不敢去海边
- 拍照时总想避开某个位置
- 别人可能觉得不明显，但自己一直很在意
- 疤痕影响过穿衣选择和身体自信

表达方向：

- 疤痕遮盖不是为了否定过去
- 也不是简单把痕迹盖住
- 而是用纹身重新整理身体上的视觉关系
- 让用户重新拥有面对身体、穿衣、拍照和生活的自由感

推荐表达：

“有些疤痕，别人可能看不太出来，但自己会在很多生活场景里反复注意到它。”

“疤痕遮盖不是简单盖住，而是让身体重新拥有被自然看见的可能。”

“用新的图案，把一直在意的位置，变成可以重新面对的一部分。”

禁止表达：

- 贩卖焦虑
- 刺激用户伤痛
- 夸张煽情
- 说疤痕毁掉人生
- 暗示不遮盖就不完整
- 承诺遮盖后完全改变人生
- 承诺完全看不出

整体语气：

温和、克制、理解、专业，有情绪价值但不过度煽情。
