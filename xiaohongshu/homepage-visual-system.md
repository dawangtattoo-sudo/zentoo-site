# ZENTOO / 针武堂刺青小红书主页视觉系统

本文件用于规范 ZENTOO / 针武堂刺青小红书主页整体视觉系统。它不是推翻现有 GitHub / Codex 工作流，也不是替代 `AGENTS.md`、`brand/`、`xiaohongshu/`、`posters/` 中已有规则，而是在现有规则基础上继续完善主页层面的视觉规划。

## 1. 主页定位

ZENTOO / 针武堂刺青小红书主页不是普通作品流水号，也不是纯品牌理念号，而是“作品编辑化呈现”的品牌主页。

核心是：

以纹身作品为主，以设计排版、文字层级、封面系统和主页整体规划做差异化。

目标是让用户感受到：

不只是纹身做得好，视觉设计、审美判断和内容呈现也很专业。

主页应让用户第一眼看到：

- 作品实力
- 设计秩序
- 空间品牌感
- 团队专业感
- 真实营业状态

## 2. 核心原则

- 作品是核心
- 设计排版是差异化
- 空间是信任
- 团队是背书
- 文案是引导
- 所有视觉设计不能喧宾夺主，不能抢作品本身
- 建议比例：70% 作品本身，30% 设计语言

执行时优先判断：

- 这张封面是否让作品更清楚，而不是让设计更抢眼？
- 这组主页是否像同一个品牌，而不是零散素材拼在一起？
- 用户是否能从主页感受到 ZENTOO / 针武堂刺青的高级、克制、现代和空间品牌感？

## 3. 主页内容类型

### 作品图文

用于展示单个作品、作品局部、身体位置关系和完成效果。作品应是第一视觉，文字只做辅助判断。

### 本周作品总结

用于整理一周内完成的作品方向。可以做合集，但不能杂乱拼贴，应统一裁切、色调和文字层级。

### 本周作品视频

用于展示一周作品的真实状态、局部质感和完成过程。视频封面必须经过选择，不能随便截帧。

### 店内工作状态视频

用于展示真实营业状态、工作台秩序、纹身师工作状态和多人纹身店的日常专业感。

### 店内空间氛围视频

用于建立到店信任。重点展示空间光线、动线、器械秩序、工作区状态和整体气质。

### 纹身手稿 / 设计过程

用于展示手稿、设计草图、身体位置关系、图案推敲和设计逻辑。重点是专业判断，不是把过程做得过度花哨。

### 单个作品审美解析

用于解释位置、比例、线条、留白、身体关系和长期观看效果。适合做收藏型图文。

### 疤痕遮盖专业内容

用于说明疤痕遮盖不是硬盖，而是重新整理皮肤上的视觉关系。必须保护隐私，不猎奇，不承诺绝对效果。

### 新手科普

用于第一次纹身、选位置、选风格、选店、沟通前准备等内容。语气应清楚、克制、有帮助，不制造焦虑。

### 品牌态度内容

用于少量情绪价值和品牌态度表达。可以使用“不要徘徊在别人的看法里”和 “TATTOO YOUR FREEDOM”，但不能鸡汤化。

## 4. 封面模板分类

### A. 纯作品型封面

适用：

- 单个作品
- 本周作品合集
- 风格合集
- 身体位置参考

特点：

- 作品大图为主
- 文字少
- 留白克制
- 右下角固定 ZENTOO / 针武堂刺青
- 不用杂乱拼贴，不用过多标签

画面建议：

- 单个作品尽量用完整身体位置关系或清晰局部
- 合集不超过 2-4 张核心图
- 作品之间保留间距，统一裁切和色调

### B. 作品解构型封面

适用：

- 位置解析
- 比例解析
- 线条解析
- 留白解析
- 身体关系解析

特点：

- 作品为主视觉
- 搭配少量注释标签
- 体现设计判断
- 注释不抢作品，不把画面做成教程大字报

画面建议：

- 可用细线、短词、编号做轻量标注
- 标签控制在 1-3 个
- 标注位置避开作品重点和皮肤细节

### C. 手稿设计型封面

适用：

- 纹身手稿
- 设计草图
- 过程图
- 身体位置关系
- 图案推敲

特点：

- 手稿或设计稿为主
- 文字少
- 专业感强
- 画面干净，不做花哨过程包装

画面建议：

- 用手稿、设计稿、参考线和身体位置关系建立专业感
- 可以搭配少量作品完成图，但不要喧宾夺主
- ZENTOO / 针武堂刺青放在边缘小字

### D. 视频封面型

适用：

- 本周作品视频
- 店内工作状态视频
- 店内空间氛围视频
- 过程记录视频

特点：

- 视频截图要经过选择，不能随便截
- 标题位置固定
- 品牌识别固定
- 视频封面也要纳入主页视觉系统

画面建议：

- 选择画面清楚、有空间感或作品质感的一帧
- 避免动作糊、表情尴尬、背景杂乱、客户身份明显
- 标题使用与图文封面相同的字体系统
- ZENTOO / 针武堂刺青固定放右下角或底部边缘

### E. 品牌态度型封面

适用：

- 品牌态度
- 情绪价值
- 审美判断
- 自由表达
- 少量主页节奏调节内容

特点：

- 大留白
- 少文字
- 空间或作品局部
- 不能鸡汤化
- 不能变成纯口号号

画面建议：

- 使用空间暗部、作品局部、安静工作状态或皮肤肌理
- 主标题 4-9 字
- 可加入“不要徘徊在别人的看法里”或 “TATTOO YOUR FREEDOM”
- 品牌态度内容应少量出现，用来调节主页节奏

## 5. 标题规则

- 封面标题默认 4-9 字
- 短、准、有判断感
- 不使用低价促销感
- 不使用江湖感
- 不使用夸张网红感
- 不要每篇都像避坑号
- 标题要服务作品和判断，不要抢作品本身

示例方向：

- 别只看图案
- 位置比图重要
- 本周作品集
- 这周做了些啥
- 手稿不是草稿
- 图好看还不够
- 店里今天很稳
- 空间也会说话
- 不必解释

标题使用建议：

- 作品型：偏简洁、审美判断
- 解构型：偏位置、比例、线条、留白
- 视频型：偏真实状态和本周记录
- 科普型：偏提醒和收藏价值
- 品牌态度型：偏短句和留白

## 6. 品牌识别规则

- 所有封面默认加入 ZENTOO
- 合适位置加入中文品牌名“针武堂刺青”
- 推荐组合：ZENTOO / 针武堂刺青
- 极简封面可以只放 ZENTOO，但正文页或最后一页必须出现“针武堂刺青”
- 品牌字样不能抢主标题，不能压住作品
- 品牌识别应作为主页秩序的一部分，而不是促销标识

推荐位置：

- 右下角小字
- 底部边缘小字
- 顶部边缘小字
- 作品留白区域边缘

不建议：

- 把品牌字样放得过大
- 压住作品主体
- 每张封面位置乱变
- 只出现英文 ZENTOO 而整组内容没有中文品牌识别

## 7. 主页九宫格节奏

建议九宫格节奏：

1. 作品图文
2. 店内工作视频
3. 本周作品视频
4. 手稿设计
5. 空间氛围视频
6. 新手科普
7. 单个作品解析
8. 品牌态度
9. 疤痕遮盖或作品合集

九宫格目标：

- 主页看起来属于同一个品牌
- 每张封面不能完全一样
- 用户能看到作品、空间、团队、过程和审美判断
- 图文和视频都纳入同一套视觉系统

统一项：

- 字体系统统一
- 品牌位置统一
- 留白方式统一
- 标题气质统一
- 色彩方向统一

变化项：

- 作品角度可以变化
- 封面类型可以变化
- 图文和视频节奏可以变化
- 作品、空间、手稿、团队状态之间要有呼吸感

建议节奏说明：

- 第 1、3、7、9 位让作品承担主页核心
- 第 2、5 位用视频建立真实营业状态和空间信任
- 第 4 位用手稿设计强化专业判断
- 第 6 位用新手科普提高收藏价值
- 第 8 位用品牌态度调节情绪和记忆点

## 8. 视频与图文统一

- 视频封面也要纳入主页视觉系统
- 视频不能随便截帧
- 视频封面标题和图文封面标题保持同一套字体和品牌识别
- 视频内容负责真实感
- 图文内容负责收藏感和设计感
- 视频和图文在主页上应形成互补，而不是两个视觉系统

视频封面选择标准：

- 画面清楚
- 主体明确
- 空间或作品有质感
- 没有客户隐私
- 没有未经授权的客户正脸
- 背景不杂乱
- 字幕和标题不遮挡作品

图文封面选择标准：

- 作品或空间为第一视觉
- 标题 4-9 字
- ZENTOO / 针武堂刺青位置稳定
- 文字层级清楚
- 不堆标签
- 适合小红书主页九宫格观看

## 9. 禁止事项

- 禁止把作品做成杂乱拼贴
- 禁止用促销风模板
- 禁止低价、优惠、限时抢等廉价转化语言
- 禁止江湖感
- 禁止过度网红化
- 禁止暴露客户隐私、电话、真实报价
- 禁止未经授权展示客户正脸
- 禁止设计元素抢过作品本身
- 禁止视频随便截帧
- 禁止标题每篇都做成焦虑型“避坑”内容
- 禁止品牌字样压住作品主体

## 10. 最终目标

让 ZENTOO / 针武堂刺青的小红书主页形成统一视觉印象：

作品实力 + 设计感 + 空间品牌感 + 团队专业感 + 真实营业状态

主页最终应让用户感受到：

- 作品是有完成度的
- 封面是有系统的
- 空间是可信的
- 团队是稳定的
- 内容是有审美判断的
- 品牌是长期、克制、现代的

这套视觉系统的重点不是把每张封面做得很满，而是让每张作品被更好地观看，让主页整体更像 ZENTOO / 针武堂刺青。
