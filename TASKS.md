# ZENTOO / 针武堂刺青任务清单

这个文件用于安排后续 Codex 工作流。每个阶段都按纹身店老板实际推进顺序整理，先把品牌和内容规则打稳，再逐步做小红书、海报、GEO、NAS、官网和 GitHub 同步。

## 第一阶段：品牌规则和内容模板完善

- 检查 `brand/zentoo-master-rules.md`，确认品牌总规则是否符合 ZENTOO 的真实气质。
- 检查 `brand/brand-principles.md`，补充品牌常用词、禁用词和文案语气。
- 固定品牌态度语：「不要徘徊在别人的看法里」。
- 固定英文表达：「TATTOO YOUR FREEDOM」。
- 整理 ZENTOO 适合长期使用的中文短句。
- 整理 ZENTOO 不适合使用的低质营销话术。
- 确认所有内容都不写客户隐私、电话和真实报价。
- 建立一份“发布前检查清单”，以后每次发内容都先检查。

## 第二阶段：小红书图文工作流

- 检查 `xiaohongshu/content-template.md`，确认模板是否适合日常发笔记。
- 按纹身风格整理小红书选题，例如黑灰、线条、东方、几何、写实。
- 按客户需求整理选题，例如第一次纹身、遮盖旧纹身、修改旧纹身、纹身位置选择。
- 制作 10 个可直接使用的小红书标题。
- 制作 5 篇小红书图文正文模板。
- 制作小红书封面文字规则，保证封面高级、清楚、不杂乱。
- 整理每篇笔记常用标签，避免乱堆标签。
- 建立每周发布计划，例如每周发几篇作品、几篇科普、几篇品牌态度内容。
- 每次发布后记录标题、发布时间、阅读量、收藏量和咨询情况。

## 第三阶段：节日海报工作流

- 检查 `posters/poster-rules.md`，确认海报风格是否符合 ZENTOO 的高级克制方向。
- 列出全年需要做海报的节日和节点。
- 先做重要节点，例如春节、情人节、五一、七夕、中秋、国庆、圣诞、跨年。
- 为每个节日准备一句主文案。
- 为每个节日准备一版克制风格海报说明。
- 建立海报尺寸清单，例如小红书封面、朋友圈图、公众号图、门店屏幕图。
- 规定海报不能出现电话、真实报价、客户隐私。
- 每张海报保存源文件和导出图，方便以后复用。

## 第四阶段：GEO推广资料工作流

- 检查 `geo/geo-keywords.md`，确认关键词是否符合真实业务。
- 补充城市、区域、商圈等本地搜索词。
- 整理 ZENTOO 的品牌关键词、品类关键词、风格关键词和审美关键词。
- 为地图平台准备一版店铺简介。
- 为官网准备一版适合搜索的品牌介绍。
- 为小红书准备一组本地搜索标签。
- 为常见搜索问题准备回答，例如“第一次纹身怎么选图”“旧纹身能不能遮盖”。
- 检查所有 GEO 文案，避免关键词堆叠和低价营销。
- 定期更新关键词，把真实咨询里常出现的问题补进去。

## 第五阶段：NAS素材管理工作流

- 检查 `nas-workflow/nas-plan.md`，确认目录结构是否适合店里实际使用。
- 在 NAS 中建立品牌、作品、小红书、海报、官网、视频、归档、私密资料目录。
- 统一文件命名方式，例如日期、类型、主题、版本号。
- 把公开素材和私密资料分开保存。
- 把客户隐私资料放入受限目录，不用于公开发布。
- 为每位纹身师建立作品归档目录。
- 为每个已发布内容保存原图、文案、封面和最终导出文件。
- 每月整理一次旧素材，把不用但需要保留的内容放入归档目录。
- 建立备份习惯，避免重要作品和品牌文件丢失。

## 第六阶段：官网首页搭建

- 检查 `website/homepage-plan.md`，确认首页结构是否适合 ZENTOO。
- 确认首页首屏主视觉使用哪类图片，例如作品细节、空间照片或工作状态。
- 准备官网首页品牌介绍文案。
- 整理作品精选图片，按风格或位置分类。
- 整理纹身师团队介绍，突出多人纹身店整体气质。
- 整理服务流程，让第一次来的客户也能看懂。
- 整理常见问题，避免写真实报价和绝对承诺。
- 设计首页的预约引导方式，但不直接写电话。
- 搭建一个基础首页版本，先保证品牌感和信息清楚。
- 检查手机端显示效果，确保文字、图片和按钮不乱。

## 第七阶段：同步到 GitHub

- 确认当前本地文件都整理完成。
- 检查是否有不该公开的内容，例如客户隐私、电话、真实报价、内部资料。
- 建立 GitHub 仓库，用来保存 ZENTOO 的公开工作流文件。
- 把本地项目初始化为 Git 项目。
- 第一次提交 README、品牌规则、模板和任务清单。
- 推送到 GitHub。
- 后续每次修改重要文件，都写清楚这次改了什么。
- 不把客户素材、私密资料、真实报价和内部敏感内容传到 GitHub。
- 定期检查 GitHub 上的内容是否仍然符合 ZENTOO 品牌规则。
